Aide
Régle: Le but pour gagner est de pousser 6 billes noires ou 6 billes blanches hors de l'hexagone.
A chaque tour, vous devez effectuer un mouvement en déplaçant de 1 à 3 billes sur l'axe de votre choix.
Vous devez pousser les billes adverses avec vos billes mais en ayant une supériorité numérique dans les déplacements avec un rapport maximum de 3 billes pour 2.
Une fois le déplacement effectué, c'est à l'adversaire de jouer.
Pour déplacer le selecteur vert sur le plateau, appuyez sur 8, 4, 5, 6 ou Z, Q, S, D.
Pour sélectionner une bille, appuyez sur 0, 7 ou B.
Une fois la bille selectionnée, déplacez vous vers la case où vous voulez aller et appuyer sur 0, 7 ou B pour confirmer le déplacement.
